My Journey
When I started to make this application, I had two goals in mind.
  • A refresher for me in JavaScript, HTML and CSS.
  • Learn Node.js although I have worked on Meteor.js which has Node.js running at it's core.

I assumed that setting up a MEAN (Mongo,Express,Angular,Node) stack was relatively easily but that turned out to be completely wrong. In fact, setting up this stack took quite some time. With Meteor, I could do it in 2 minutes. Yes, development was that fast. :)

Challenges that I faced during this setup were,

  • Setting up the required node modules.
  • Exploring template engines like EJS, Jade and Angular.
  • Understanding the intricacies of Express.js and render the first frontend page from the Node.js backend.
  • Forming HTTP requests from AngularJS to Express.js APIs retrieve and update documents in mongo database.
  • Making both Express.js routing and Angular routing work together.
  • Handling template arrays in AngularJS.
  • Configuring hosting to start with Nodemon for the Node.js app on Heroku.

Considering the minimal knowledge I had in Angular and no experience in Node.js, Express.js, mlab and Heroku, I think the outcome is prety decent. :)
